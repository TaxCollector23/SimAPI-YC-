## Install

```bash
pip install simapi
```

## Configure

```bash
export SIMAPI_BASE_URL="https://sim-api.vercel.app/api"
export SIMAPI_API_KEY="sk_live_..."
```

## Validate

```python
import simapi

result = simapi.validate(
    data="cfd_output.csv",   # path, list[dict], DataFrame, or ndarray
    simulation_type="aerodynamics",
    conditions={"velocity": 15.0, "altitude": 120.0},
)

print(result.status)            # "passed"
print(result.confidence)        # "high"
print(result.training_ready)    # True
print(result.summary())         # pretty report
```

## Inspect the result

<CodeGroup>

```python Stats by column
# Statistics are attribute-accessible by canonical name
print(result.drag_coefficient)        # StatResult(mean=0.312, std=0.018, n=196)
print(result.drag_coefficient.mean)   # 0.312
df = result.to_dataframe()            # all stats as a DataFrame
```

```python Checks
for check in result.failed_checks():
    print(check.name, check.detail)

for check in result.warning_checks():
    print("⚠", check.name)
```

```python Export
result.download_csv("clean_stats.csv")
```

</CodeGroup>

## Accepted input types

<ParamField path="data" type="str | list | DataFrame | ndarray">
  A file path (CSV/JSON/NPY), a list of dicts, a pandas `DataFrame`, or a NumPy
  array. Everything is converted to records before submission.
</ParamField>

## Convenience helpers

```python
simapi.demo()                 # validate seeded synthetic data
simapi.health()               # server facts
simapi.get_job("3f9ca12b")    # fetch a prior result
```

<Warning>
  The SDK raises `RuntimeError` with the structured error `code`, `message`, and
  `request_id` on non-2xx responses — log the `request_id` for support.
</Warning>
